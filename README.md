# YS WebP Tools

YANGSHEEP DESIGN 圖片優化外掛 — 上傳自動轉 WebP、自動縮圖、關閉不需要的縮圖尺寸，降低圖片體積、節省磁碟空間、加快網站載入。

## 功能

| 功能 | 說明 |
|------|------|
| **WebP 轉換** | 新上傳的 JPG/PNG 自動轉成 WebP。預設取代原檔（最省空間），可開啟「保留原始檔」並存。品質與來源格式可設定。 |
| **自動縮圖** | 上傳超過設定最大寬/高的圖片時，自動等比例縮小並覆寫原圖。 |
| **縮圖尺寸管理** | 自動偵測所有已註冊的縮圖尺寸（含主題／外掛動態尺寸與 WP 大圖 `-scaled` 門檻），可逐項或**一鍵全部**停用。另提供**批次重新產生**工具，把設定套用到既有圖片（刪除停用尺寸的舊縮圖、重建啟用的尺寸）；偵測到 WooCommerce 時引導至其內建重生工具。 |

> 本外掛僅處理「啟用後新上傳」的圖片，**不會變動既有媒體庫**。

## 安裝

1. 將 `ys-webp-tools` 目錄上傳至 `wp-content/plugins/`
2. 於「外掛」頁面啟用「YS WebP Tools」
3. 進入後台「電商工具箱 → WebP 工具」進行設定

需求：WordPress 6.0+、PHP 8.0+，伺服器需支援 GD 或 Imagick 的 WebP 功能。

## 設定

後台「電商工具箱 → WebP 工具」分為五個頁籤：

1. **一般設定** — 外掛總開關
2. **WebP 轉換** — 啟用開關、品質（1–100）、來源格式（JPEG/PNG/GIF）、是否保留原始檔
3. **自動縮圖** — 啟用開關、最大寬度、最大高度（0 = 不限）
4. **縮圖尺寸管理** — 自動偵測尺寸列表、逐項或一鍵全部停用、批次重新產生既有圖片縮圖、WooCommerce 引導
5. **說明文件**

### 安全預設

所有具破壞性的功能（WebP 取代原檔、縮圖覆寫原圖）**預設關閉**，需於設定頁明確開啟後才生效。安裝啟用外掛本身不會改動任何檔案。

## 注意事項

- 「取代原檔」會刪除原始 JPG/PNG，無法復原；若需保留請開啟「保留原始檔」。
- GIF 動圖轉 WebP 會失去動畫，因此 GIF 預設不勾選轉換。
- 自動縮圖排除 GIF，避免破壞動畫。
- 支援 JPG／JPEG／PNG（`.jpg` 與 `.jpeg` 同屬 image/jpeg，皆可轉換）。
- 「批次重新產生縮圖」會刪除既有縮圖後重建，無法復原，執行前建議備份。

## 技術資訊

- 命名空間：`YangSheep\WebpTools`（PSR-4，`src/`）
- 設定儲存於自訂資料表 `{prefix}ys_webp_tools_settings`（非 wp_options）
- 後台操作走 AJAX（nonce + 權限驗證），非 form POST
- HPOS 相容（WooCommerce 非必要依賴）
- 透過 YS Plugin Hub Client 提供自動更新
